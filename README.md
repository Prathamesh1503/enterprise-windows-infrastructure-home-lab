# Enterprise Windows Infrastructure Home Lab

## Overview

The Enterprise Windows Infrastructure Home Lab is a hands-on project designed to simulate a real-world enterprise IT environment using virtualization.

The objective is to gain practical experience in Windows Server administration, enterprise networking, Active Directory, and IT support while following industry best practices for deployment, documentation, and troubleshooting.

This project is being built step by step and documented throughout the implementation process.

---

## Objectives

- Build an enterprise Windows infrastructure from scratch
- Learn Windows Server Administration
- Configure Active Directory Domain Services (AD DS)
- Configure DNS and DHCP
- Manage users, groups, and Organizational Units (OUs)
- Implement Group Policy Objects (GPOs)
- Configure File and Print Services
- Practice Windows administration and troubleshooting
- Develop technical documentation and diagrams
- Build a professional portfolio project for IT Support and Cybersecurity roles
